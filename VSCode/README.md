# VSCode

## プロジェクトの構成

| file                            | description                                                  |
| ------------------------------- | ------------------------------------------------------------ |
| `.vscode/extensions.json`       | プロジェクトでの導入推奨拡張機能を記述するファイル           |
| `.vscode/settings.json`         | プロジェクトの設定ファイル                                   |
| `<project_name>.code-workspace` | ワークスペースの設定ファイル                                 |
| `pyproject.toml`                | Pythonプロジェクトの設定を一元管理するための標準設定ファイル |
| `requirements.txt`              | Pythonで使用するライブラリを記述するファイル                 |
